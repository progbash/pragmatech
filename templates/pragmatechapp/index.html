Tədris paketləri

Foundation

Harvard Universitetinin CS50 kurrikulumu əsasında hazırlanmış tədris metodudur. Bu tədris modulu daxilində tələbələr böyük problərin alqoritmlər vasitəsilə asanlıqla həll oluna biləcəyini, proqramlaşdırmanın ümumi fəlsəfəsini və proqramların arxa planda necə işlədiyini çox əyləncəli tətbiqlər vasitəsilə öyrənəcəklər.

HTML & CSS

Verilən dizaynı çox sürətli şəkildə HTML formatına çevirə bilmək üçün tələb olunan bütün HTML və CSS funksionallığı ilə yanaşı tələbələrə kod yazma sürətlərini artıra biləcək əlavə konsepsiyalar, framework-lər və kodlama üslubları öyrədilir.

Javascript

ES6 standartları çərçivəsində Javascript-in dil olaraq bütün xüsusiyyətləri əhatə olunur. Bundan əlavə Javascript kitabxana istehsal etmə məntiqi, Javascript frameworklərinin (əsasən React) işləmə fəlsəfəsi və tətbiqi, eyni zamanda bu dilə məxsus köməkçi alətlərdən istifadə olunması öyrədilir.

C#

Obyekt yönümlü paradiqmanın ən yaxşı tətbiq olunduğu dillərdən olan C# orta və böyük miqyaslı tətbiqlərin yazıla bilməsi üçün çox münbit bir mühit təmin edir (.NET platforması). Bundan əlavə Microsoft-un yeni tətbiq etməyə başladığı multiplatformalı Core konsepsiyası ilə fərqli platformlar üçün tətbiqlər yazıla bilər.

PHP

Açıq qaynaqlı və multiplatforma dəstəkli dil olan PHP öyrənməsi rahat, geniş dakumentasiyaya sahib bir dildir. Tələbələr bu dil vasitəsi ilə rahatlıqla veb layihələr yaza bilmə qabiliyyətinə və PHP-nin Laravel və digər framework-lərinin işləmə prinsiplərinə dərindən yiyələnəcəklər.

Python

Python hal hazırda dünyada ən çox öyrənilən dillərdən biridir. Bunun səbəbi Python-un çox geniş əhatədə tətbiq edilə bilməsidir. Tələbələr Python Core öyrənməklə və veb sahəsində tətbiq etməklə bu dilin digər sahələrdə də istifadəsinə hazır hala gələcəklər.

Sual-Cavab

Təhsil üçün ödədiyin məbləği Pragmatech tərəfindən sənə veriləcək işlərin yerinə yetirilməsi qarşılığında 100% geri qazanacaq və ya işlə təmin olunma zəmanəti əldə edəcəksən.

İmtahanlar həm nəzəri həm də praktiki imtahanlar formasında olacaq.

Foundation modulunu bitirdikdən və tələb olunan imtahanı uğurla keçdikdən sonra sənə verilən işləri təhvil verməklə maddi gəlir əldə etməyə başlayacaqsan.

Oxuduğun və dinlədiyin məlumatı asanlıqla başa düşə bilməyin bizim dərsləri rahatlıqla qavraman üçün kifayətdir.

Təhsil modulları sənin tamamlamalı olduğun təhsil planlarının tərkib hissəsidir və hər biri ayrı mövzuları əhatə edir.Təhsil planı isə sənin Pragmatech daxilindəki təhsilin üçün yol xəritəsidir.

Sizin təhsil keyfiyyətiniz Pragmatech tərəfindən təmin olunur. Tədris modullarının hər biri fərqli müəllimlər tərəfindən tədris oluna bilər. Ona görə də sizin hansı müəllimdən dərs alacağınız təhsil prosesi başladıqdan sonra müəyyən olunacaq.

Ödənişlər aylıq ödəmə formasında olacaq və hər ayın əvvəli ödənilir.

Dərslər Azərbaycan dilində tədris olunur. Tədris zamanı istifadə olunan vəsaitlər ingilis dilində olacaq. Buna görə də ingilis dili biliyiniz əhəmiyyətlidir.
